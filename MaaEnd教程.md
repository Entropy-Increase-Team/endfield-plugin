# MaaEnd 远程控制教程

远程控制 MaaEnd Client，实现终末地自动化任务管理。

## 前置要求

- 下载 [MaaEnd](https://maaend.com)
- 下载 [MaaEnd Client](https://github.com/Entropy-Increase-Team/MaaEnd-Client/releases)

## 快速开始

### 1. 安装 MaaEnd Client

1. 将下载好的 MaaEnd Client 放入 MaaEnd 文件夹下
2. 双击运行即可

### 2. 绑定设备

1. **私聊**机器人发送 `:maa 绑定`，获取绑定码
2. 在 MaaEnd Client 中输入绑定码完成设备绑定
3. 发送 `:maa 设备` 确认设备已上线

### 3. 设置默认设备

绑定完成后，设置默认设备以省略后续命令中的设备序号：

```
:maa 设置设备 1
```

### 4. 查看可用任务

```
:maa 任务列表
```

### 5. 执行任务

```
:maa 执行 daily        # 按任务名执行
:maa 执行 1 2 3        # 按序号执行多个任务
```

任务下发后会自动附带一张设备截图，任务完成后也会推送通知和截图。

## 命令一览

### 设备管理

| 命令 | 说明 |
|------|------|
| `:maa 绑定` | 生成绑定码（仅私聊） |
| `:maa 设备` | 查看已绑定设备列表 |
| `:maa 设置设备 <序号>` | 设置默认设备，后续命令可省略序号 |
| `:maa 删除设备 <序号>` | 解绑并删除设备 |

### 任务操作

| 命令 | 说明 |
|------|------|
| `:maa 任务列表 [序号]` | 查看设备可用任务（省略序号则使用默认设备） |
| `:maa 执行 <任务名或序号>` | 在默认设备上执行任务 |
| `:maa 状态 [编号]` | 查询任务状态（默认最近一次，支持 1~20） |
| `:maa 停止 [编号]` | 停止任务（默认最近一次，支持 1~20） |
| `:maa 历史 [页码]` | 查看任务执行历史 |

### 设备控制

| 命令 | 说明 |
|------|------|
| `:maa 截图 [序号]` | 获取设备当前截图（省略序号则使用默认设备） |
| `:maa 重置 [序号]` | 重置设备任务状态（设备卡住时使用） |

> **提示**：所有命令支持 `:` `：` `#zmd` `#终末地` 四种前缀。
